Library for Data Analysis

Spectrum
  • get()
    This function retrieves the spectrum data from the source.
  • plot()
    This function plots the spectrum data.
  • zoom_plot()
    This function allows you to zoom into a specific region of the plot.
  • zoom_cycle_plot()
    This function cycles through different zoom levels in the plot.
  • scatter_plot()
    This function generates a scatter plot of the spectrum data.
  • data_loader()
    This function loads the spectrum data from external sources.
  • denoise_spectrum()
    This function removes noise from the spectrum data.
Linecut
  • get_cut()
    This function retrieves a line cut from the dataset.
  • plot()
    This function plots the line cut data.
  • denoising()
    This function applies denoising techniques to the line cut.
  • one_peak()
    This function fits a single peak to the line cut data.
  • two_peak()
    This function fits two peaks to the line cut data.
  • three_peak()
    This function fits three peaks to the line cut data.
  • fit()
    This function performs a fit to the line cut data using a predefined model.
